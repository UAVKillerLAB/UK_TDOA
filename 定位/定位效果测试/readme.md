### 2. 项目源码简要说明
- 1.代码原理
    基于基站坐标（866，500），（-866，500），（0，-1000），随机输入一个信号源坐标，生成时差，再运用定位算法进行定位测试定位效果
- 2.效果说明
    在当前情形之下，若信号源位于三站以内或靠近三站范围附近，那么定位精度相当高，最大误差不超过2米，测试出来的平均误差在0.5m以内；
    若信号源位于三站以外较远，那么定位精度相当低，想准确定位是不可能的。
